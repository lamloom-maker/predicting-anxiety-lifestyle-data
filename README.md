# Predicting Anxiety from Lifestyle Data

This repository presents an individual predictive analytics project focused on predicting anxiety levels using lifestyle and behavioral data.

## Project Overview
The goal of this project is to analyze lifestyle-related features and build predictive models to identify patterns associated with anxiety.

## Objective
To explore how lifestyle and behavioral factors contribute to anxiety levels using predictive analytics and machine learning techniques.

## Dataset
The dataset includes demographic, lifestyle, and behavioral variables related to anxiety levels.

**Note:** The dataset is not included in this repository due to privacy and licensing constraints.

## Methodology
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Feature engineering  
- Predictive modeling  
- Model evaluation  

## Models Used
- Logistic Regression  
- Machine Learning classifiers  
- Evaluation using accuracy and performance metrics  

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Scikit-learn  
- Matplotlib / Seaborn  
- Jupyter Notebook  

## Results
The models demonstrated the ability to identify meaningful patterns related to anxiety levels, showing promising performance during evaluation.

## Project Type
Individual Academic Project

## Repository Structure
- `notebooks/` : Analysis and modeling notebooks  
- `src/` : Python scripts  
- `docs/` : Final project report
